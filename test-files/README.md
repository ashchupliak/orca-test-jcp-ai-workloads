# IDE Test Files

Test the IDE by:
1. Opening these files in the IDE
2. Editing them
3. Running the JavaScript file: `node hello.js`
4. Testing VS Code features (syntax highlighting, autocomplete, etc.)

When you run `node hello.js`, you should see:
```
Hello from Orca Lab IDE!
Hello, Developer!
Doubled: [ 2, 4, 6, 8, 10 ]
Async functions work!
Success!
```
